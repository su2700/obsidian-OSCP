
```
for file in *.json; do jq . "$file" > "$file.tmp" && mv "$file.tmp" "$file"; done
```

## 每一部分在干什么？

### 1️⃣ `for file in *.json; do`

- 遍历当前目录下 **所有 `.json` 文件**
    
- 每次循环：
    
    - `file` = 一个 JSON 文件名
        

---

### 2️⃣ `jq . "$file"`

- `jq` = JSON 处理工具
    
- `.` = **原样输出，但会自动格式化**
    
- 相当于：
    
    - 校验 JSON 是否合法
        
    - 并把它 **漂亮地缩进**
        

---

### 3️⃣ `> "$file.tmp"`

- 把格式化后的结果：
    
    - 写入临时文件 `原文件名.json.tmp`
        
- ❗**不直接覆盖原文件**（安全）
    

---

### 4️⃣ `&& mv "$file.tmp" "$file"`

- `&&` 表示：
    
    - **只有当前一步成功才执行下一步**
        
- 如果 `jq` 成功：
    
    - 用临时文件 **替换原 JSON 文件**
        

📌 如果 JSON **格式错误**：

- `jq` 失败
    
- `mv` 不会执行
    
- 原文件 **不会被破坏**
    

---

## ✅ 最终效果（一句话）

> **批量美化当前目录下所有 JSON 文件，并安全覆盖原文件**
